# 基本概念

组成：内圈、外圈、滚动体、保持架。

圆柱：线接触、速度低、承载高

滚针：减小轴承尺寸。

保持架：隔离滚动体，减少相邻滚动体的磨损。

内圈与轴颈配合。

外圈与轴承座孔装配。

在滚动过程中，受径向载荷作用下产生内部轴向力 $F_a$,导致内外圈有分离的趋势，需要成对使用

也就是说一压滚动体就会产生一种轴向力。

内部轴向力与角度 $\alpha$ 有关。

分为：向心轴承(径向)、推力轴承(轴向)、向心推力轴承(面对面和背对背安装)。

## 轴承代号

前置代号(基本没用)、基本代号、后置代号

### 基本代号

**轴承内径**：右起第一第二数字，一般是5的倍数，代号是内径尺寸的关系。

特殊的 ：00 01 02 03，分别代表10、12、15、17mm。

**直径系列**：基本代号右起第三个数字，即内径相同外径和宽度变化。

**宽度系列**：右起第四个数字，没有特殊要求就是0，可以省略。

**轴承类型代号**：右起第五个数。

其中，直径系列和宽度系列统称为尺寸系列代号。

### 后置代号

**内部结构代号**：表示同一类轴承不同的内部结构，主要有：C 15°、AC 25°、B 40°。

公差等级代号：由高到低，如P0、P6、P5、P4、P2。

径向游隙系列：1组、2组、0组、3组、4组、5组，由小到大，其中0组可以不标注。

## 向心轴承
  - 角接触: $\alpha$ = 0°，主要径向力，可以小轴向力。
  - 径向接触：$\alpha$ = 0到45°，同时承受轴、颈向力。

## 推力轴承
- 推力角接触轴承： $\alpha$ = 45°到90°，主要轴向力，可以小径向力。
- 轴向接触轴承：$\alpha$ = 90°，只能径向力。
