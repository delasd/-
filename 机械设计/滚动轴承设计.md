# 基本概念

组成：内圈、外圈、滚动体、保持架。

圆柱：线接触、速度低、承载高

滚针：减小轴承尺寸。

保持架：隔离滚动体，减少相邻滚动体的磨损。

内圈与轴颈配合。

外圈与轴承座孔装配。

在滚动过程中，受径向载荷作用下产生内部轴向力 $F_a$,导致内外圈有分离的趋势，需要成对使用

也就是说一压滚动体就会产生一种轴向力。

内部轴向力与角度 $\alpha$ 有关。

分为：向心轴承(径向)、推力轴承(轴向)、向心推力轴承(面对面和背对背安装)。

## 轴承代号

前置代号(基本没用)、基本代号、后置代号

### 基本代号

**轴承内径**：右起第一第二数字，一般是5的倍数，代号是内径尺寸的关系。

特殊的 ：00 01 02 03，分别代表10、12、15、17mm。

**直径系列**：基本代号右起第三个数字，即内径相同外径和宽度变化。

**宽度系列**：右起第四个数字，没有特殊要求就是0，可以省略。

**轴承类型代号**：右起第五个数。

其中，直径系列和宽度系列统称为尺寸系列代号。

### 后置代号

**内部结构代号**：表示同一类轴承不同的内部结构，主要有：C 15°、AC 25°、B 40°。

公差等级代号：由高到低，如P0、P6、P5、P4、P2。

径向游隙系列：1组、2组、0组、3组、4组、5组，由小到大，其中0组可以不标注。

## 向心轴承
  - 角接触: $\alpha$ = 0°，主要径向力，可以小轴向力。
  - 径向接触：$\alpha$ = 0到45°，同时承受轴、颈向力。

## 推力轴承
- 推力角接触轴承： $\alpha$ = 45°到90°，主要轴向力，可以小径向力。
- 轴向接触轴承：$\alpha$ = 90°，只能径向力。

## 轴承的选择

轴承的选择一般从四个方面考虑：承载能力、轴承转速、调心能力、安装拆卸

- 承载能力

  - 载荷大或冲击大时，选滚子轴承；
  - 载荷小或冲击小时，选球轴承；
  - 纯径向载荷：深沟球或者圆柱滚子；
  - 纯轴向：止推轴承；
  - 既有轴向力又有径向力：圆锥滚子；

- 轴承转速
  - 高速轻载选球轴承，低速中再选滚子轴承。
  - 在高速时，宜选用相同内径而外径较小的轴承。
  - 外径较大的轴承，宜用于低速重载的场合；

- 调心能力
  - 由于滚子轴承对偏斜最为敏感，所以当有一定轴线偏斜时应避免采用。

- 安装拆卸
  - 轴承座没有剖分面而必须沿轴尚装拆时，优先选用内、外圈可分离的轴承；
 
## 失效形式
- 疲劳点蚀
- 塑性变形
- 磨损
### 计算准则
- 对一般转速（n>10r/min）轴承，疲劳点蚀是主要失效形式，应进行寿命校核计算；

- 对静止或极慢转速（n≤10 r/min）轴承，承载能力取决于所允许的塑性变形，应进行静强度计算;

- 对高速轴承，除进行寿命计算外，还应进行极限转速校核计算。

## 轴承装置的设计
轴承装置的设计主要解决安装、配置、紧固、调节、润滑、密封等问题

1. 安装

安装轴承的壳体和轴承座有足够的刚度；另外在一根轴上两个支撑座孔，要保持必要的同心度。

2. 轴承的配置
  - 双支撑点各单向固定。用于温度不高的短轴。<br>这种轴承配置常用两个反向安装的角接触球轴承或圆锥滚子轴承。
    - a). 故前者悬臂较长，支承刚性较差。
    - b). 在受热变形方面，因运转时轴的温度一般高于外壳的温度，产生变形容易卡死；
    - 此支撑方式也可用在深沟球轴承上。
  -一支点双向固定，另一端支点游动；用于跨度较大且工作温度较高的轴。 

## 轴承的调节
- 轴承的游隙和预紧可以靠端盖下的垫片来调整。

- 1. 锥齿轮或蜗杆在装配时，为了便于调整轴向位置，可将确定其轴向位置的轴承装在一个套杯中，套杯则装在外壳孔中，通过增减套杯端面与外壳之间垫片的厚度，调整锥齿轮或蜗杆的轴向位置。

- 2. 两端游动支撑
<br>由于人字齿轮轴中，为了防止轴向卡死和人字齿两侧受力不均的现象出现。需使小齿轮轴做成游动，故两端游离支撑。
- 3. 轴承的紧固
  - 用周用弹性挡圈前入轴的沟槽内，主要用于轴向力不大及转速不高的场合。
  - 用螺钉固定的轴端挡圈紧固用于高速旋转且承受轴向力较大的场合。
  - 用圆螺母和止动垫圈固定用于轴承高速旋转承受轴向力较大的情况。
  - 用紧定衬套止动垫圈和圆螺母紧固，用于光轴、轴向力和转速都不大的，内圈为圆锥孔的轴承。
 
## 润滑及密封
### 润滑
润滑的目的：降低摩擦阻力，散热，减少接触应力，吸收振动，防止锈蚀等。

- 脂润滑：在滚动轴承的速度较低时长采用脂润滑滚动轴承的装脂量一般为轴承内部空间的1/3到2/3为宜。
- 油润滑：在高温高速的条件下常采用油润滑，滚动轴承常用油润滑的方式有油浴润滑、滴油润滑、溅油润滑、喷油润滑等。其中飞溅润滑是一般闭式齿轮传动装置中
 ### 密封
密封的目的：防止润滑剂从轴承中流失，防止灰尘，水分，酸气等外界杂质进入轴承。

- 接触式密封：毡圈密封，唇形密封圈密封和密封环密封。
<br>毡圈密封主要用于脂润滑的场合，唇型密封圈密封唇的方向要朝向密封的部位。
- 非接触式密封：间隙密封、甩油密封和曲路密封。
