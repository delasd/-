# 沿程损失( $h_f$ )和局部损失( $h_m$ )

| 名称 | 定义 | 特点 | 计算公式 |
| -------- | -------- | -------- | -------- |
| 沿程损失 | 沿程不变的管段上具有沿程阻力，克服沿程阻力引起的能量损失叫沿程损失 | 沿程损失沿管段均布，与管段的长度成正比，也叫长度损失 | 沿程水头损失： $h_f = \lambda \frac{l}{d} \cdot \frac{v^2}{2g}$ <br>压强损失： $p_f = \rho g h_f$|
| 局部损失 | 在边界急剧变化的区域存在集中分布的阻力，叫局部阻力,克服局部阻力引起的能量损失叫局部损失。| 引起局部损失的原因是由于旋涡区的产生和速度方向和大小的变化。 | 局部水头损失: $h_m = \xi \cdot \frac{v^2}{2g} $<br>压强损失: $p_m = \rho g h_m$ |

整个管路的能量损失等于各管段的沿程损失和局部损失的总和。

# 层流、紊流、雷诺数
## 两种状态

| 名称 | 定义 | 特点 |
| -------- | -------- | -------- |
| 层流 | 各液层毫不相混，分层有规则的流动状态叫做层流。 | 只存在黏性引起的各层间滑动摩擦阻力。|
| 紊流 | 液体质点的运动轨迹极不规则，各部分流体互相剧烈掺混的流动状态叫做紊流。 | 除了黏性阻力，还存在由于质点掺混，互相碰撞所造成的惯性阻力。|

层流 --> 紊流 为上临界流速，紊流 --> 层流 下临界流速($v_k$)

默认所指临界流速为下临界流速

## 临界雷诺数

雷诺数(判断流态)：

$$ R_e = \frac{v d\rho}{\mu} = \frac{v d}{\nu} $$

临界雷诺数(200 -- 4000)：

$$ R_{eK} = \frac{v_k d}{\nu} = 2000 $$

其中：
v 是流体流速

$\nu$ 是运动粘度

$\mu$ 是动力粘度

## 流态分析

流态取决于绕动的惯性作用和黏性的稳定作用斗争的结果(谁是主导作用)

## 层流底层

在紊流的流态中，在流管边缘存在极小的区域内存在层流，由于固体壁面的阻滞作用，流速小，惯性力小，依然是层流作用。

存在一个过渡层，紊流核心过渡导层流底层。

随着雷诺数增大，过渡层减小。

# 圆管中层流运动

水力坡度(单位长度的沿程损失)：

$$ J = \frac{h_f}{l} $$

轴向切应力 $\tau$ 与J之间的关系：

$$ \tau = \rho g \frac{r}{2} J $$

圆管均匀流中，切应力与半径成正比，在断面上按**直线规律**分布，轴线上为0，管壁上最大。

沿程阻力系数：

$$ \lambda = \frac{64}{R_e} $$

与管壁的粗糙度无关，只和雷诺数有关。

在圆管层流中，流速是以管中心为轴线的旋转抛物面，即**抛物线规律**。

平均流速等于最大流速的一半：

$$ v = \frac{1}{2} u_{max} $$

# 紊流运动的特征和紊流阻力
## 运动特征

脉动现象：诸如速度、压强等空间点上的物理量随时间的变化作无规则的即随机的变
动。（时均法统计)

紊流脉动的强弱程度用**紊流度ε**表示。

## 紊流阻力

紊流阻力包括黏性切应力和惯性切应力：

| 名称 | 定义 |
| -------- | -------- |
| 黏性切应力 | 因紊流中各流层间的相对运动而产生 |
| 惯性切应力（雷诺应力） | 因脉动引起的动量交换所产生 |

## 混合长度理论(半经验理论)

雷诺数越大，紊动越剧烈，黏性切应力的影响就越小:

层流时的切应力：黏性切应力

紊流时的切应力：惯性切应力

# 尼古拉兹实验
## 沿程阻力系数以及影响因素分析

沿程阻尼系数 $\lambda$ ---- $R_e$ 和壁面粗糙(相对粗糙度)

| 名称 | 定义 |
| -------- | -------- |
| 层流 | 层流阻力 --- 粘性阻力 $\lambda = \frac{64}{R_e}$  $\lambda$ 仅与 $R_e$ 有关 |
| 紊流 | 紊流阻力 --- 黏性阻力、惯性阻力； $\lambda$ 和 $R_e$ 、壁面粗糙有关 |

相对粗糙度：

$$ \frac{K}{d} 或 \frac{K}{r_0} $$ 

相对光滑度：

$$ \frac{d}{K} 或 \frac{r_0}{K} $$ 

| 名称 | 范围 | 特点 |
| -------- | -------- | -------- |
| 层流区 | $R_e$ < 2000  | $\lambda$ 随 $R_e$ 的增大而减小，与K/d 无关 |
| 临界区 |  $R_e$ = 2000 - 40000 | $\lambda$ 随 $R_e$ 的增大而增大，与K/d 无关  |
| 紊流光滑区 |  $R_e$ > 4000 | $\lambda$ 仅与 $R_e$ 有关，与K/d 无关  |
| 紊流过渡区 |  | $\lambda$ 既与 $R_e$ 有关，又与K/d 有关  |
| 紊流粗糙区(阻力平方区) |  | $\lambda$ 只与K/d 有关，与 $R_e$ 无关  |

# 非圆管的沿程损失

水力半径：

$$ R = \frac{A}{X} $$

沿程损失和水力半径成反比

X ：过流断面上流体固体壁面接触的周界。(周界越短，水力半径越大)

当量直径:

$$d_e = 4R $$

|  | 圆管 | 矩形断面 | 正方形断面 |
| -------- | -------- | -------- | -------- |
| 水力半径 | $\frac{d}{4}$ | $\frac{ab}{2(a+b)}$ | $\frac{a}{4}$ |
| 当量直径 | | $\frac{2ab}{(a+b)}$ | a |

当量直径在非圆管的雷诺数计算，沿程损失计算、沿程阻力系数计算中相当于圆管直径，但不能用于连续性方程。

# 管流动的局部损失
## 局部损失的一般分析

局部损失的计算公式：

$$ h_m = \xi \frac{v^2}{2g} $$

局部损失的阻力系数与雷诺数成反比：

$$ \xi = \frac{B}{R_e} $$

### 局部阻碍

#### 突变

**漩涡区**：在边壁突变的地方，出现主流与边壁脱离的现象，主流与边壁之间形成漩涡区(漩涡区越大，能量损失越大)

#### 渐变

**渐扩管**：减速增压区，在一定的 $R_e$ 范围内，漩涡区的位置及大小与 $R_e$ 有关。

**渐缩管**：减压增速区。不会出现漩涡区。

局部阻碍的影响长度：在局部阻碍范围内损失的能量，只占局部损失的一部分，另一部分是在局部阻碍下游一定长度的管段上损耗的。

## 减少阻力的措施


**两种途径**：

1.改进流体外部的边界

2.在流体内部投加极少量的添加剂，使其影响流体运动的内部结构。

---

减小紊流局部阻力的着眼点在于防止或推迟流体与壁面的分力，壁面漩涡区的产生或减小漩涡区的大小和强度。

(1)管道进口：平顺的管道进口

(2)渐扩管和突扩管：较小的扩散角

(3)弯管：弯管阻力系数在一定范围内随曲率半径R增大而减小。

(4)三通：减小支管与合流管之间的夹角，或将连接处的折角改缓。

(5)合理衔接：先弯后扩不合理
