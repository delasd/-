# 傅里叶变换

非周期信号可以看作周期信号 $T_0$ 趋于无穷大的周期信号。

周期信号的频谱间隔无线接近，变为连续谱。

经过变换后，x(t) 函数变为 x($\omega$)

$$ x(\omega) = \int^{\infty}_{\infty} x(t) e^{-j \omega t} dt $$

$$ x(t) = \frac{1}{2\pi} \int^{\infty}_{\infty} x(\omega) e^{j \omega t} d \omega $$

令 $\omega = 2 \pi f$ 带入得：

$$ x(f) = \int^{\infty}_{\infty} x(t) e^{-j 2 \pi f t} dt $$

$$ x(t) = \int^{\infty}_{\infty} x(f) e^{j 2 \pi f t} d f $$

# 能够进行傅里叶积分的条件是：

- 1.在有限区间上满足狄利赫里条件；

- 2. $\sum^\infty_{-\infty} |x(t)| dt$ 收敛，即绝对可积。
 

# 非周期信号频谱的特点

基频无限小，即o趋近于0。

频谱连续。

$|X(\omega)|$ 与 $|C_n|$ 量纲不同。$|C_n|$ 具有与原信号幅值相同的量纲，$|X(\omega)|$ 是单位频宽上的幅值。

# 傅里叶变换的主要性质

1.奇偶虚实性

当x(t)为实数时，可得：

实部为偶函数，虚部为奇函数。

此时，频谱只剩下实部，故为偶函数

因此



| 时域 f(t) 的对称性 | 频域 F(ω) 的对称性 | 频域的实/虚部表现 |
| ------------ | ------------ | --------- |
| 实函数          | 共轭对称         | 实部偶，虚部奇   |
| 实偶函数         | 实偶函数         | 纯实、偶      |
| 实奇函数         | 纯虚奇函数        | 纯虚、奇      |
| 虚函数          | 共轭反共轭对称      | 实部奇，虚部偶   |
| 虚偶函数         | 虚偶函数         | 纯虚、偶      |
| 虚奇函数         | 实奇函数         | 纯实、奇      |
| 复函数（一般）      | 无特殊对称        | 实虚部皆无奇偶要求


2. 线性叠加性
 
3. 时频对称性

频域图像与时域图像的变换是相互的，
X(t) -- x(-f)

4. 尺度改变性

$$ x(kt) = \frac{1}{k}X(\frac{f}{k}) $$

如果时域扩展，频域就会收缩；

如果频域收缩，时域就会扩展。

5. 时移性

时域中平移，就等于频域中的旋转。

平移只改变相频频谱，不改变幅值谱。

6. 频移特性

若像函数频移，则时域乘以负正的e的次方。

7.卷积特性

两个时域信号的卷积傅里叶变换等于其各自傅里叶变换的乘积。

两个频域信号的卷积傅里叶变换等于其各反自傅里叶变换的乘积。

# 典型的信号频谱

## 单位脉冲信号

性质：

1. 乘积性(采样性)

$$ x(t)(t\pm t_0) = x(\bm t_0)(t\pm t_0) $$

2. 筛选性

$$ \int x(t)(t\pm t_0) = \int x(\bm t_0)(t\pm t_0) $$

可以筛选x(t)的取值

3. 卷积性

卷积就等于x(t)的本身。

$$ x(t) * (t\pm t_0) = x(t\pm t_0) $$

将函数图像进行搬迁。
