# 分类

按信号随时间变化的特征分：

- 确定信号和不确定信号

按信号幅值随时间变化的连续性分：

- 连续信号和离散信号

按信号的能量特征分：

- 能量信号和功率信号

按信号的分析域不同分：

- 时域信号和频域信号

# 确定性信号和非确定性信号
信号：
- 确定性信号
  - 周期信号
    - 谐波信号
    - 一般周期信号
  - 非周期信号
    - 准周期信号
    - 瞬态信号
- 非确定性信号
  - 平稳随机信号
    - 各态历经信号
    - 非各态历经信号
  - 非平稳随机信号

# 确定信号

明确数学关系式或图像表达的信号

- ## 周期信号

  以一定的时间间隔周而复始出现，x(t) = x(t + nT)

  - ### 谐波信号

    频率单一的正弦或余弦信号

    均方根：

    均方值：

  - ### 一般周期

    多个乃至无穷多个单周期信号叠加，而且存在公共周期。方波、三角波。

    判断方法：各信号频率或周期比为有理数。

- ## 非周期信号

  - ### 准周期信号

    由多个乃至无穷多个单周期信号叠加，但是不存在公共周期。

判断方法：若干信号中，至少有两个信号的频率之比不是有理数。

### 瞬变信号

在有限时间存在，随时间增加幅值衰减至0。

如单位脉冲函数。



# 非确定性信号

又叫随机信号，无明确的数学关系式。

根据是否满足平稳随机过程的条件分为

- ## 平稳随机信号
  统计特征不随时间变化
- ## 非平稳随机信号
  随时间变化。

特点：不重复性。

不确定性。

不可预估性。

# 连续和离散信号

- 连续信号[模拟信号（信号的幅值与独立变量均连续）
  - 一般连续信号（独立变量连续）
- 离散信号
  - 一般离散信号（独立变量离散）
  - 数字信号（信号的幅值与独立变量均离散)
 
 # 能量信号和功率信号
在其分析区域内，能量为有限值为能量信号。

如，各类瞬变信号。

如果能量信号无限，但是有限区间做功是有限的，为平均功率有限信号。

如，周期信号、准周期信号、随机信号。

对于随机信号，常用**功率谱**进行分析。

# 时域信号和频域信号

 - ## 时域描述
   以时间为独立变量；反映信号的幅值随时间变化的关系

  - ## 频域描述
    以频率为独立变量，由信号的时域描述通过适当方法变换得到；反映信号的频率结构和各频率成分的幅值、相位关系。

两种方法是在不同角度对信号进行描述的。


# 信号的时域描述

以时间为独立变量，描述信号随时间的变化特征，

反映信号幅值随时间变化的关系。

波形图：时间为横坐标的幅值变化图。

优点：形象、直观。

缺点：不能明显揭示信号的内在结构（频率组成关系）

# 信号的频域描述

以频率为独立变量建立信号幅值、相位与频率的函数关系。

频谱图：以频率为横坐标的幅值、相位变化图。

- 幅值谱：幅值-频率图

- 相位谱：相位-频率图

频域描述抽取信号内在的频率组成及其幅值和相角的大小，
描述更简练、深刻、方便。
